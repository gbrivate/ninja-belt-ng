Desafios - Angular/Java

:: Desafio 1

  1. Angular: Criar uma rota no Angular usando o próprio nome do time, ex. */teamA, ao carregar a rota o componente tem que implementar um serviço que vai chamar um REST feito pelo time de Java.
  2. Java: Criar um serviço POST:/team/'teamName' que vai cadastrar o time e os participantes do mesmo, detalhe, um mesmo time não pode ser cadastrado mais de uma vez.
  3. Java: Criar um serviço GET:/team/list que retorne todos os times/partipantes.
  4. Angular: Listar os dados do time e do Presenters, aplicando filtros assim evitando que dados de outro time seja mostrado.(Não será considerada a solução que mostrar dados de outros times).
