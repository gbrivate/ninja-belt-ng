Desafios - Angular/Java

:: Desafio 1

  1. Angular: Criar uma rota no usando o próprio nome do time, ex. */teamA, ao carregar a rota, o componente tem que implementar um serviço que vai chamar um REST POST:/team/'teamName'.
  2. Java: Criar um serviço POST:/team/'teamName' que vai cadastrar o time e os participantes do mesmo, detalhe, um mesmo time com o mesmo nome, não pode ser cadastrado mais de uma vez.
  3. Java: Criar um serviço GET:/team/list que retorne todos os times/partipantes.
  4. Angular: Listar os dados via GET:/team/list do time e do Presenters, aplicando filtros, assim evitando que dados de outro time seja mostrado.(Não será considerada a solução que mostrar dados de outros times).

:: Desafio 2

  1. Angular: Chamar o REST POST:/belt/'teamName' via serviço e listar os belts do time, os belts terá que ser mostrado num modal e invocado através de um link/button.
  2. Java: Criar um serviço REST POST:/belt/'teamName' que retorne a lista belts do time, (Não será considerado serviço que retorne cinturões de outros times.)
